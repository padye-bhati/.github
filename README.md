Bhati Script Design (Based on Devanagari)

Guiding Principles

Minimal inventory: Only keep letters used in Bhati dialect.

Phonemic clarity: One symbol per sound.

No conjunct complexity: Avoid stacked ligatures.

Oral-first cadence: Script should match spoken Bhati rhythm.

Vowels

अ (a)

इ (i)

उ (u)

ई (ī)

ऊ (ū)

ए (e)

ओ (o)

◌ॅ (short /æ/, essential for Bhati words like घरॅ)

Nasalization: ◌ँ (vowel nasalization), ◌ं (generic nasal)

Removed: ऋ, ॠ, ऌ, ॡ, औ (rare in Bhati speech)

Consonants

Core: क, ग, त, द, प, ब, म, न, च, ज, य, र, ल, व, स, ह

Optional (if aspirates are distinct): ख, थ, फ

Retroflex (minimal use): ट, ड

Removed: श, ष (merge into स), Persian/Arabic letters (ज़, फ़, etc.)

Diacritics

◌ॅ for /æ/ endings (बरॅ, घरॅ)

◌ँ for nasalized vowels (गावें)

Virama (्) only for final consonants, avoid conjunct stacks

Orthography Rules

Always write vowels explicitly — no hidden schwa deletion.

Use ◌ॅ for Bhati endings like बरॅ, घरॅ.

Mark nasalization clearly (सग्ळीं).

Avoid conjunct ligatures — linear writing preferred.

Simplify sibilants — use स for all /s/ sounds.

Examples from Bhati Text

गतकां → Sayings/verses.

Conclusion

This minimal Bhati script subset of Devanagari captures the dialect’s unique phonetics and oral rhythm. It simplifies unused letters, emphasizes nasalization and the /æ/ vowel, and provides a practical orthography for writing Bhati poetry, proverbs, and everyday speech.
